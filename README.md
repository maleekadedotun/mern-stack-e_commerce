# 🛒 MERN E-commerce Platform

## 🚀 Overview
A full-featured e-commerce web application built with the MERN stack. This platform allows users to browse products, add items to cart, make secure payments using Stripe, and receive order confirmations. It also includes an admin dashboard for managing products, users, and orders.

---

## 🛠️ Tech Stack
- Frontend: React.js, Redux Toolkit
- Backend: Node.js, Express.js
- Database: MongoDB
- Payment Gateway: Stripe
- Image Hosting: Cloudinary
- Authentication: JWT (JSON Web Token)

---

## ✨ Features

### 👤 User Features
- User registration & login (JWT authentication)
- Browse products with search & filter functionality
- Add to cart & manage cart items
- Secure checkout with Stripe payment gateway
- Order history & tracking
- Responsive design for mobile & desktop

---

### 🛍️ Product Features
- Product listing with images (Cloudinary)
- Categories & filtering
- Product details page
- Ratings & reviews (optional)

---

### 💳 Payment Integration
- Secure payments using Stripe
- Payment confirmation & order creation
- Error handling for failed transactions

---

### 🛠️ Admin Features
- Admin dashboard
- Create, update, delete products
- Upload product images via Cloudinary
- Manage users
- Manage orders & update order status
